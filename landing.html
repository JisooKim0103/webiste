About

KDT Cloud Navigator is a system built for easy and fast way of searching files in a organization this website also have file tagging and categorizing features that helps the user to arrange its file.

Our services

Search Engine

A program that searches for and identifies items in a database that correspond to keywords or characters specified by the user, used especially for finding particular sites on the World Wide Web.

File Tagging

File tagging is a way of organizing files or folder by applying tags to them. This tag can then be used to search for data, and sort the files accordingly. With this feature, you can tag files into categories irrespective of their location.

File Categorizing

A file classification scheme is the process of identifying the category or categories of business activities and the records they generate and grouping them, if applicable, to facilitate retrieval, description, control, links and also for determining their disposition and access status.
